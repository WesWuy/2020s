# 2020s: The Board Game

**Prototype v0.15 — Comic-Broadcast UI Pass**

A satirical survival board game and browser game prototype where players race from **2020 to 2030** while trying not to lose their **Sanity**, **Money**, or **Influence**.

> Survive the decade without losing your mind, your money, or your grip on reality.

## Art Direction

```text
https://weswuy.github.io/2020s/art-direction.html
```

The art pass establishes the visual language: **satirical retro-futurist news broadcast meets comic-book apocalypse**. It adds reusable character portrait slots, event art frames, lower-third broadcast styling, halftone texture, Dystopia / Utopia color direction, and prompt guidance for generated artwork. As of v0.15, AI-generated character portraits and a broadcast hero panel fill those slots.

## Playtest Goal

Run a rough 20–45 minute test. For the board game, judge clarity, humor, balance, and shareability. For the video game mode, judge whether the choice loop, visual identity, and Dystopia / Utopia split feel fun enough to become a real browser/mobile game.
