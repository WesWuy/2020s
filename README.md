# 2020s: The Board Game

**Prototype v0.3 — Digital Playtest Assistant**

A satirical survival board game where players race from **2020 to 2030** while trying not to lose their **Sanity**, **Money**, or **Influence**.

> Survive the decade without losing your mind, your money, or your grip on reality.

## Live Prototype Site

```text
https://weswuy.github.io/2020s/
```

## Digital Playtest Assistant

```text
https://weswuy.github.io/2020s/play.html
```

The assistant is a hot-seat browser prototype. It can choose players, assign characters, roll a die, move around the 40-space board, draw cards, track stats, switch Red Cycle / Blue Cycle, keep a turn log, and export playtest notes.

## Playtest Goal

Run one rough 30–45 minute game with 3–4 players. Do not judge visual polish yet. Judge whether the game is clear, funny, balanced, and worth playing again.
