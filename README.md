# 2020s: The Board Game

**Prototype v0.13 — Video Game Feel Pass**

A satirical survival board game and browser game prototype where players race from **2020 to 2030** while trying not to lose their **Sanity**, **Money**, or **Influence**.

> Survive the decade without losing your mind, your money, or your grip on reality.

## Video Game Feel Pass

The v0.13 pass makes the browser game feel more like a real game. It adds character preview cards, active character cards, chapter cards, mode-colored backgrounds, event categories, previous-choice feedback, Timeline Health status labels, and stronger Dystopia / Utopia identity.

## Public Playtest Readiness

The v0.11 readiness pass makes the board game easier to hand to strangers. It adds Quick Start Chaos, a 60-second rules panel, Mild / Unhinged / Apocalyptic chaos levels, card tags for balance testing, known playtest notes, and direct feedback links for bugs, card ideas, playtest reports, and balance issues.

## Playtest Goal

Run a rough 20–45 minute test. For the board game, judge clarity, humor, balance, and shareability. For the video game mode, judge whether the choice loop, visual identity, and Dystopia / Utopia split feel fun enough to become a real browser/mobile game.

## Version Roadmap

- **v0.11:** public playtest readiness with quick start, rules summary, chaos tuning, card tags, known notes, and feedback links.
- **v0.12:** video game mode foundation with Dystopia Run, Utopia Run, choice events, Timeline Health, and final timeline reports.
- **v0.13:** video game feel pass with character identity, event categories, outcome feedback, and mode-specific visual styling.
- **v1.0:** public preview build.
