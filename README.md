# 2020s: The Board Game

**Prototype v0.12 — Video Game Mode Foundation**

A satirical survival board game and browser game prototype where players race from **2020 to 2030** while trying not to lose their **Sanity**, **Money**, or **Influence**.

> Survive the decade without losing your mind, your money, or your grip on reality.

## Video Game Mode

```text
https://weswuy.github.io/2020s/video.html
```

The new video game foundation adds two browser-playable runs: **Dystopia Run**, where the goal is to survive the decade, and **Utopia Run**, where the goal is to improve Timeline Health while surviving. Each run uses character stats, year-by-year event scenes, player choices, Timeline Health, run logs, and final timeline reports.

## Public Playtest Readiness

The v0.11 readiness pass makes the board game easier to hand to strangers. It adds Quick Start Chaos, a 60-second rules panel, Mild / Unhinged / Apocalyptic chaos levels, card tags for balance testing, known playtest notes, and direct feedback links for bugs, card ideas, playtest reports, and balance issues.

## Content Explosion Pack

The v0.10 content pack expands the decks from 65 to 110 prototype cards. It adds public table-action prompts, audience votes, conspiracy corkboard moments, monologue challenges, forbidden-phrase style prompts, expansion-pack energy, stronger endgame cards, new achievements, and more ridiculous ending titles.

## Playtest Goal

Run a rough 20–45 minute test. For the board game, judge clarity, humor, balance, and shareability. For the video game mode, judge whether the choice loop feels fun enough to become a real browser/mobile game.

## Version Roadmap
